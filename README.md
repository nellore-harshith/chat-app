# 💬 Chat Application

A simple real-time chat application built using React.

## 🚀 Features

* Send and receive messages
* Timestamp for each message
* Delete message (for me & for everyone)
* Pin important messages
* Clean and modern UI
* Data persistence using localStorage

## 🛠 Tech Stack

* React.js
* JavaScript
* HTML & CSS

## 📦 Setup Instructions

1. Clone the repository:
   git clone https://github.com/nellore-harshith/chat-app.git

2. Navigate to project folder:
   cd chat-app

3. Install dependencies:
   npm install

4. Run the app:
   npm start
